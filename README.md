# Computer Science Study

### 📕 About
- 신입 개발자가 알아야 할 필수 기초 CS를 공부하고 기록하는 저장소입니다.

<br>

### 📝 How to
- 스터디는 주 2회, `월 / 목, 13:00` 에 진행합니다.
- 커리큘럼 하위 주제 1개를 선정하여 학습 내용을 해당 레파지토리에 정리합니다.
- 발표자 1인을 선정하여 본인이 학습한 내용을 발표하고, 질의응답을 나눕니다.
- 부족한 내용 또는 오류 사항 등을 점검하고 피드백하여, 정확한 개념을 학습합니다.

<br>

### ⚠️ Penalty
- 불참 시 벌금 3,000원, 학습 내용 미정리 시 벌금 2,000원이 부과되며, 예외는 없습니다.
- 벌금은 매달 말일에 `1/N` 로 송금합니다.
- 카카오뱅크 모임 계좌로 투명하게 관리합니다.

<br>
