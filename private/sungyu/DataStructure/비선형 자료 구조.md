# 비선형 자료 구조  

- 요소가 일렬로 나열되지 않고, 자료 순서나 관계가 복잡한 자료 구조를 의미  
  ex) 트리, 그래프 등  

<br>

### 그래프 (Graph)  
> 정점(노드)과 이를 연결하는 간선으로 이루어진 자료구조  

<img width="423" alt="image" src="https://github.com/user-attachments/assets/92a3db70-1233-4b72-92d6-ff98e9af4b1b" />

<br>

**구성 요소**  
- 정점 (Vertex) / 노드 (Node)  
- 간선 (Edge)  
  - 정점 간의 관게를 표현할 수 있다.  
  - 간선에 값을 부여해 가중치를 나타낼 수 있다.  
  - 방향성을 가질 수 있다.  

<br>

**관련 알고리즘**  
- BFS (Breadth First Search)  
- DFS (Depth First Search)  

<br>
<br>

### 트리  
> 그래프의 한 종류로 정점과 간선으로 이루어진 계층적 데이터 집합  

<img width="515" alt="image 2" src="https://github.com/user-attachments/assets/4ae79681-7072-4549-81f6-b9a2f49669aa" />

<br>

**이진 트리** (Binary Tree)  
> 자식 노드의 수가 두 개 이하인 트리  

<img width="1786" height="504" alt="image 3" src="https://github.com/user-attachments/assets/42ffd94e-2c0e-4212-bc5e-1bab2c1c3ec8" />

- **정이진 트리** (Full Binary Tree) : 자식 노드가 없거나 2개인 이진 트리  
- **완전 이진 트리** (Complete Binary Tree) : 마지막 레벨을 기준으로, 왼쪽부터 채워진 이진트리  
- **변질 이진 트리** (Degenerate Binary Tree) : 자식 노드가 하나밖에 없는 이진트리  
- **포화 이진 트리** (Perfect Binary Tree) : 모든 노드가 꽉 차있는 이진 트리  
- **균형 이진 트리** (Balanced binary Tree) : 왼쪽과 오른쪽 노드의 높이 차이가 1 이하인 이진트리  

<br>

**이진 탐색 트리** (Binary Search Tree)  
> 노드의 왼쪽 하위 트리에는 ‘노드보다 작은 값’, 오른쪽 하위 트리에는 ‘노드보다 큰 값’이 포함된 트리  

- 노드의 위치에 따라 값의 범위가 정해져있기 때문에 검색(탐색)에 용이하다.  

일반적인 탐색 : O(logn)  
최악의 경우 탐색 (= 선형적인 경우) : O(n)

### 힙 (Heap)  
> 완전 이진 트리 기반 자료구조  

**최대 힙**  
- 루트 노드에 있는 키가 모든 자식 노드의 키들 중 가장 큰 값을 갖는다.  

<img width="545" alt="image 5" src="https://github.com/user-attachments/assets/585eedb8-74da-4475-b6d3-f561f89178c7" />

**최소 힙**  
- 루트 노드에 있는 키가 모든 자식 노드의 키들 중 가장 작은 값을 갖는다.  

데이터 삽입 /삭제 : **O(log n)**  
탐색 : **O(1)**  

<br>

### 우선순위 큐 (Priority Queue)  
> 힙을 이용하여 구현되며, 큐의 특징을 가지고 있지만 먼저 들어온 순서가 아닌, 우선순위를 기준으로 나가는 형태의 자료구조  

```python
import heapq

pq = []

# 삽입
heapq.heappush(pq, (3, "A"))
heapq.heappush(pq, (1, "B"))
heapq.heappush(pq, (5, "C"))


# 삭제 (최소값)
print(heapq.heappop(pq))  # (1, "B")


# 조회
print(pq[0])  # (3, "A")
```

### 딕셔너리 (Dictionary)  
> Key와 매핑된 Value의 조합으로 형성된 자료구조  

```python
student = {
    "name": "홍길동",
    "age": 25,
    "major": "컴퓨터공학"
}

print(student["name"])  # 홍길동
```

<br>

### 집합 (Set)  
> 수학의 집합 개념과 동일한 자료형  

**특징**  
- 순서가 없다.  
- 중복을 허용하지 않는다.  

```python
A = set()
A.add(2, 3, 4, 4, 5)

print(A)	# A = {2, 3, 4, 5}
```

<br>

### 해시 테이블  
> 수 많은 데이터들을 해시 값으로 매핑한 테이블  

데이터 삽입 /삭제 : **O(1)**  
탐색 : **O(1)**
